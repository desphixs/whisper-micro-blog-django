# Build Whisper: A Secure Micro Blog

### Project Description

You are going to build Whisper, a secure micro blogging timeline where users can post short messages. You will start by creating a database model that links every new message to the exact user who wrote it. Then, you will build a global feed where logged in users can see everyone's thoughts in real time. After that, you will write custom security logic to make sure users can only edit or delete their own posts, and hide those buttons from strangers. Finally, you will build custom profile pages so people can click on a user and see a history of only their specific messages. You will write everything using raw Django code and function based views so you understand the exact flow of data and security.
